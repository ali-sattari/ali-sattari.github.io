The Site

I first started blogging in 2004 in Persian, mostly about the Linux, front-end development, and web standards (it was the fad back then among cool kids), this continued to 2009 when I last published a post.

I have been meaning to start a new blog in English for a while but I was so tired of setting up a CMS and fiddle with it. I wanted something different, and Hugo combined with Github Pages gave me that different thing: fewer hassles, better control, and minimum costs.
